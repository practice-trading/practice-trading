# Learn and Practice Trading by Doing

Welcome to the official GitHub profile repository for **ChartingPark**.

ChartingPark is a structured trading practice platform designed to help aspiring and intermediate traders build real skill through hands-on experience — not passive consumption.

---

## What Is ChartingPark?

ChartingPark is where traders:

- Practice trading on real market data
- Build risk management discipline
- Improve decision-making through repetition
- Track performance metrics like win rate, drawdown, and expectancy
- Compete on skill-based leaderboards

The focus is simple:

**Learn trading by doing.**
